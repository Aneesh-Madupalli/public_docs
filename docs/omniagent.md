# 🔐 Privacy Policy — OmniAgent

**Effective Date:** March 18, 2026
**Product:** OmniAgent — Multi-LLM Sidebar AI Agent Platform (SaaS + Chrome Extension)

---

## 1. Introduction

OmniAgent (“we”, “our”, “us”) is an AI-powered platform that enables users to interact with multiple large language models (LLMs) through a browser sidebar and perform actions using an agent + tool system.

This Privacy Policy explains how we collect, use, process, and protect your data when you use:

* Our Chrome Extension
* Our Web Platform (authentication, settings, billing)
* Our APIs and backend services

By using OmniAgent, you agree to this policy.

---

## 2. Information We Collect

### 2.1 Account & Authentication Data

When you register or log in:

* Email address
* Authentication credentials (securely handled via providers)
* Account metadata (plan, usage limits)

---

### 2.2 User Inputs & AI Interactions

To provide core functionality, we process:

* Prompts you send to AI models
* Responses generated by AI systems
* Tool execution inputs and outputs

⚠️ These are processed in real-time to deliver features such as chat, summarization, and automation.

---

### 2.3 Browser Context Data (Extension)

OmniAgent accesses limited webpage data **only when required for a feature**:

* Selected text (when user explicitly selects)
* Visible page content (for summarization, extraction)
* Metadata (e.g., page title, URL)

🚫 We do NOT:

* Continuously track browsing history
* Collect background browsing activity

---

### 2.4 Tool-Specific Data

When using tools (e.g., YouTube analyzer, data extractor):

* Relevant page data is processed
* Outputs (e.g., summaries, structured data) are generated

---

### 2.5 API Keys (Optional)

If you provide your own API keys:

* Stored in encrypted format
* Used only to connect to selected LLM providers
* Never exposed publicly

---

### 2.6 Usage & Analytics Data

We collect limited operational data:

* Request counts
* Feature usage (tools, models)
* Error logs and performance metrics

Used strictly for improving system performance and reliability.

We use collected data to:
