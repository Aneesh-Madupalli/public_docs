# Privacy Policy

This privacy policy explains how this browser extension handles user data.

## Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

The extension interacts with web pages only to detect HTML5 video elements and adjust video playback speed based on user actions such as keyboard shortcuts or extension controls. No website content, browsing history, or user activity data is collected or recorded.

## Data Usage

All functionality is performed locally within the user’s browser. User preferences such as playback speed, indicator settings, and toggle options are stored locally using the browser’s storage APIs and never leave the user’s device.

## Website Access

The extension requires access to websites in order to detect and control HTML5 video playback. This access is used solely to modify video playback speed and does not involve reading, collecting, or transmitting page data.

## Third-Party Services

This extension does not use analytics services, tracking technologies, cookies, advertising networks, or third-party data processors.

## Data Sharing

No user data is sold, transferred, or shared with third parties under any circumstances.

## Security

The extension does not communicate with external servers. All scripts are bundled with the extension package and executed locally within the browser.

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.

## Contact

If you have any questions about this privacy policy, please contact the developer through the extension’s support page.
