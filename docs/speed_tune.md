# Privacy Policy

This document explains how this browser extension handles user data.

## Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

## Data Usage

The extension operates entirely on the user’s device. All functionality is performed locally within the browser.

User preferences such as settings or configuration options are stored locally using the browser’s storage APIs and are never transmitted outside the user’s device.

## Third-Party Services

This extension does not use analytics services, tracking technologies, cookies, advertising networks, or any third-party data processors.

## Permissions

Permissions requested by the extension are used solely to provide its core functionality and are not used to access, collect, or monitor user data beyond what is required for that functionality.

## Data Sharing

No user data is sold, transferred, or shared with third parties under any circumstances.

## Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.

## Contact

If you have questions regarding this privacy policy, please contact the developer through the extension’s support page.
